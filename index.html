Ce service vous permet de partager un mot de passe à une utilisation à n'importe qui de manière sécurisés

Security

Nous stockons vos mot de passe de manière sécurisés grâce à un algorithme de chiffrement. Le message est chiffrer dans votre navigateur avant d'être envoyé à la base de donnée. Une fois le message lu, le texte est effacé de notre base de donnée.

Comment créer un bon mot de passe ?

Un bon mot de passe doit respecter certaines règles de compléxité

  • Au minimum 8 caractères
  • Des minuscules
  • Des majuscules
  • Des chiffres
  • Des caractères spéciaux (symboles)

De plus il est conseillé de faire un mot de passe différent par site

Utilisation Unique

Plutôt que d'envoyer des mots de passes par mail, message, bout de papier. Il est préférable d'utiliser des lien unique à la place. Une fois le message lu, le message est automatiquement détruit
