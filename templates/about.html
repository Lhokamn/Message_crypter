About Message crypter

Welcome to the Message crypter, this app allow you to send sensitive data at anyone. Created by Corentin Claudel, this app is a fully open-source project aimed at promoting transparency, collaboration, and user freedom.

This app is licensed under the GNU General Public License v3.0 (GPL-3.0). The GPL ensures that the software remains free and open for anyone to use, modify, and distribute, under the same licensing terms.

Message crypter is part of the wider Open Source Initiative (OSI), an organization that promotes and protects open-source software by advocating for software freedom. By choosing open source, we aim to empower users and developers to contribute to, improve, and share the project without limitations.

We prioritize the security of your sensitive information. When sharing confidential data, we strongly encourage the use of encryption tools. This ensures that your messages are protected from unauthorized access.

For enhanced privacy, we support the use of encrypted messages that can only be read once. After the first viewing, the message is automatically deleted, ensuring that your sensitive data remains safe and cannot be accessed again.

Key Features

  • Completely free and open-source
  • Community-driven improvements
  • Customizable to fit different workflows

Advantages

  • message is encrypted in database
  • After one week, message is erased from database

Framework

Message crypter is based on python app, and sqlite3 databases
